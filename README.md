# rime-japanese

## 输入习惯调整
+ 模糊音nn → n （nn → n → ん）
+ 模糊音x → _ （xa → _a → ぁ）

## About

This is a layout for typing in Japanese 日本語. Supports words in all 3 scripts (Kanji, Hiragana, Katakana).


## Installing

Edit `default.custom.yaml` and add `japanese` to the schema list:

```bash
patch:
  schema_list:
    - schema: japanese
```

Reload RIME.
